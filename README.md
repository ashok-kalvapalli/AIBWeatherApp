# AIBWeatherApp
AIBWeatherApp Sample


How to build

    Clone the repository

$ git clone https://github.com/ashok-kalvapalli/AIBWeatherApp.git

    Install pods - Alo

$ cd AIBWeatherApp
$ pod install

    Open the workspace in Xcode

$ open "AIBWeatherApp.xcworkspace"
